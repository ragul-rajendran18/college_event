# EVENTRA - College Event Registration System

A modern, responsive frontend for college event management built with Vite, React, and Tailwind CSS v4.

## Features

- 🚀 **Lightning Fast**: Powered by Vite and Tailwind CSS v4 for optimized performance.
- 🎨 **Modern UI**: Sleek glassmorphism design with vibrant gradients and smooth animations.
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop viewports.
- 🔍 **Event Discovery**: Features event searching and category filtering.
- 📋 **Registration System**: Comprehensive form validation and submission feedback.
- ✨ **Enhanced UX**: Uses `framer-motion` for smooth page transitions and hover effects.

## Tech Stack

- **Build Tool**: Vite
- **Frontend**: React 19
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7

## Getting Started

1. **Install Dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components (Header, Footer, EventCard, etc.)
- `src/pages`: Main application views (Home, EventDetail)
- `src/data`: Mock event data for demonstration
- `src/utils`: Utility functions (e.g., tailwind class merging)
- `src/index.css`: Global styles and design system tokens
